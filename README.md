# Clive Bell – Civilization (1928) – Visualisierungen

Eine Sammlung interaktiver Processing-Skripte, die Clive Bells Essay *Civilization* (Pelican Books, 1938-Ausgabe) visuell interpretieren.  
Inspiriert von Bells These: **Zivilisation ist kein Selbstzweck, sondern ein Mittel zu guten Geisteszuständen**.

## Über das Projekt

Clive Bell argumentiert in seinem Essay, dass nur **Geisteszustände** intrinsisch wertvoll sind. Zivilisation (Arbeit, Gesetze, Reichtum, Sicherheit) existiert nur, um diese Zustände (ästhetische Ekstase, Kontemplation, Liebe, intellektuelle Klarheit) zu ermöglichen.

Dieses Repository übersetzt zentrale Ideen des Buches in **interaktive Kunst-Animationen** mit Processing.

### Bisher umgesetzte Ideen (Stand Januar 2026)

1. **Propaganda-Plakat** (satirisches WWI-Poster)  
2. **Marschierende Slogans** (endlose Kolonne mit Propaganda-Texten)  
3. **Krieg vs. Zivilisation Kollision** (Zerstörung von Büchern/Statuen)  
4. **Tatort-Fadenkreuz** (Civilization im Visier)  
5. **Timeline der Kriegsziele** (Entwicklung von Justice → Civilization)  
6. **Morphende Propagandaposter** (Kandinsky-Transformation)  
7. **Karussell aus Buchseiten** (Bilder aus Ordner laden)  
8. **Spiralleiter zu guten Geisteszuständen** (Leiter/Spirale + Bröckeln)  
9. **Werkzeugkasten → Transformation in Kunst** (Zivilisation wird Kunst)  
10. **Bewusstseinswolken über Zivilisationslandschaft** (Wolken steigen auf)  
11. **Kontrast-Maschine** (schlechte vs. gute Zustände)  
12. **Portal der Muße** (von Not zur Kontemplation)

Alle Skripte nutzen gemeinsame Features:
- Zoom (MouseY, 33%–300%)
- Pan-Modus ('z')
- Auto-Zoom-Puls ('q')
- Farbmodi-Wechsel ('w')
- Neue Farbpalette ('c')
- Screenshot ('s')
- Reset ('r')
- Neues Hintergrundbild ('l')
- Pause ('p')

### Installation & Ausführung

1. **Processing installieren** → https://processing.org/download
2. Dieses Repository klonen oder herunterladen
3. Einen Ordner `data/images/` anlegen und Bilder (z. B. PDF-Seiten) hineinlegen
4. `particle_texts.txt` mit deinen Zitaten/Texten füllen
5. Skript öffnen → Run (Strg+R)

### Struktur

- `/sketches/` → alle .pde-Dateien  
- `/data/` → Texte & Bilder  
- `/docs/` → ausführliche Kapitel-Dokumentation (zukünftig)  
- `/assets/` → Screenshots & Cover
